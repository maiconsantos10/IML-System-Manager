# 🧾 IML System Manager

## 🧠 Sobre o projeto

O **IML System Manager** é um sistema desenvolvido em Java com o objetivo de simular a gestão de registros administrativos em um Instituto Médico Legal (IML).

O projeto foi inspirado em rotinas reais de sistemas internos, com foco em organização de dados, estruturação de código e simulação de funcionalidades utilizadas no dia a dia de sistemas corporativos.

---

## 🎯 Objetivo

Este projeto foi criado com foco em:

* Praticar conceitos de **Programação Orientada a Objetos (POO)**
* Simular a construção de um sistema real do zero
* Evoluir gradualmente para integração com banco de dados
* Desenvolver raciocínio de back-end

---

## 🏗️ Estrutura do projeto

O projeto segue uma organização baseada em separação de responsabilidades:

```id="tlfq0u"
src/
 ├── application/   # Inicialização do sistema (Main)
 ├── entities/      # Classes de domínio (modelos)
 ├── database/      # Conexão e inicialização do banco
 ├── ui/            # Interface gráfica (Swing)
```
